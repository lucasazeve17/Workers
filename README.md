##  💻 Sobre o projeto

Aplicação que visa a divulgação de serviços,fornecendo facilidade ao usuario de achar o serviço desejado, seja ele designer,pedreiro,buffet e etc. 
## 🛠️ Tecnologias
mobile:
 - React native
 - Expo
 - Axios

backend:
 - Node.Js
 - Sequelize
 - mySql
 - JWT
 - Express
